# 首页落地页设计

## 概述

为未登录用户提供产品介绍落地页，引导注册/登录。已登录用户继续看到现有仪表盘。

## 路由策略

- `/` 路径根据登录状态展示不同内容
  - 未登录 → 落地页（产品介绍 + 引导注册）
  - 已登录 → 仪表盘（现有学习概况页面）

## 页面结构

### 1. Hero 区
- 大标题："掌握词汇，从科学记忆开始"
- 副标题：简述 SM-2 间隔重复的核心价值
- CTA 按钮："免费开始" → 注册页，"已有账号" → 登录页
- 背景：渐变色 + 浮动单词气泡动画

### 2. 交互演示区
- 可翻转的单词卡片（模拟真实学习体验）
- 用户可点击翻转看释义
- 说明文字："点击卡片，体验翻转学习"

### 3. 核心特性区（3 列网格）
- SM-2 间隔重复算法 - 科学安排复习时间
- 多种学习模式 - 翻转卡片 / 拼写测试 / 选择题
- 学习数据追踪 - 进度可视化，成就激励

### 4. 底部 CTA 区
- "准备好开始你的词汇之旅了吗？"
- 注册按钮

## 布局处理

- 落地页不显示 Navbar 和 MobileNav（全屏沉浸式）
- 已登录仪表盘保持现有布局不变

## 技术方案

- 在 `page.tsx` 中根据 `useSession` 状态条件渲染
- 落地页提取为独立组件 `LandingPage`
- 仪表盘提取为独立组件 `Dashboard`
- 使用 CSS 动画实现卡片翻转和浮动效果
- 布局层面通过 session 状态控制 Navbar/MobileNav 显隐
